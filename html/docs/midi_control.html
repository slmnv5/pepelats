MIDI control


MIDI input control selection

Looper searches for MIDI inputs with specific names provided in configuration file. If nothing is found looper computer keyboard is used to send MIDI commands. If computer keyboard is not connected looper starts anyway but may not be controlled. Look at log file via looper's web page to check if MIDI input is connected.

Auto restart if MIDI control is missing

Looper will auto restart if no any control received over time period. If not a single message was received since the start it likely means nothing is connected and looper will restart in 1 minute. If some messages were received before but then stopped the looper will restart in 10 minutes, even if a song is playing or there is unsaved content.

MIDI input controller configurations

;midi_in_names -- names (or part of names) for MIDI input controls, case-sensitive
midi_in_names : BlueBoard, FootCtrlPlus

;keyboard_keys -- keys on computer keyboard if using keyboard as MIDI IN
keyboard_keys : 1, 2, 3, 4, q, w

;keyboard_notes -- MIDI notes (or CC) sent by each button, integers from 0 to 127
keyboard_notes : 60, 62, 64, 65, 12, 13

Notes are labeled: A, B, C, D, E, F, G, H... and they correspond to integer values in keyboard_notes option in the [main.ini]
