Configuration

All configuration (except of user licence) is saved in file [user.ini] that has two sections [looper] and [audio] that define configuration of looper and audio device respectively.

Every user needs to adjust this file to match equipment used. If [user.ini] is empty or lack specific setting then default file [main.ini] is used.

You may edit [user.ini] in the web page of the looper. Thus, for example, you may change names of MIDI or audio devices, MIDI notes used and so on. If you need example of specific configuration look at [main.ini] that contains all default configurations. There are comments – lines starting with semicolon – that explain each option in each section.


Sections of configuration files

These configuration files have two sections in [audio] and [looper]. It is important to keep each configuration in correct section. For an example you may always refer to [main.ini]


Basic MIDI configuration

Basic MIDI configuration is saved in [user.ini] / [main.ini] has three things:


  • input MIDI port names separated by comma, part of name may be used instead of full name and names are case-sensitive, first matching port is used for MIDI input

  • Six MIDI notes (or CC numbers) device may send. This is list of integers from 0 to 127. Default values are: 60, 62, 64, 65, 12, 13

  • Six typing keyboard keys that may be used when MIDI controller is missing. Default values are: 1, 2, 3, 4, q, w
