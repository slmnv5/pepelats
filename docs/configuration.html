Looper configuration

File [user.ini] has two sections [looper] and [audio] that define configuration of looper and audio device respectively.

Edit this file to change default configurations. Thus, you may change names of MIDI or audio devices, MIDI notes used and number of MIDI notes used.

If you need example of specific configuration look at [main.ini] that contains all default configurations. There are comments – lines starting with semicolon – that explain each option in each section.
