Configuration

All configurations except of user license are saved in two files:

  • [user.ini] - user configuration
  • [main.ini] - default configuration

If file user.ini does not have specific configuration it is taken from file main.ini. You may dit file user.ini in the main web page of the [looper].
Thus, for example, you may change names of MIDI or audio devices, MIDI notes used and so on. To see an example of specific configuration look at [main.ini] that contains all default configurations and comments explaining each option.

MIDI configuration

MIDI configuration in [user.ini] / [main.ini] has only three option:


  • input MIDI port names separated by comma, part of name may be used instead of full name and names are case-sensitive, first matching port is used for MIDI input
  • Six MIDI notes (or CC numbers) device may send. This is list of integers from 0 to 127. Defaults are: 60, 62, 64, 65, 12, 13
  • Six computer keyboard keys to use for MIDI messages. Defaults are: 1, 2, 3, 4, q, w (see [Installation] to use computer keyboard)
