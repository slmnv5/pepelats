Menu layout


Menu layout translates MIDI notes into looper commands. The layout used is defined in local configuration file [user.ini].

If this file is missing menu layout it is taken from standard configuration file [main.ini].

These two files define input MIDI port name(s), number of buttons, MIDI notes (or CC) these buttons send and typing keyboard keys that may be used when MIDI controller is missing.

The menu layout is found in directory: [menu](../config/menu), there are 3 INI files:

  • [play.ini] -- view for playing and recording of song parts and loops

  • [song.ini] -- view for managing songs and drums

  • [serv.ini] -- view for restart and other service functions

Each view may have several sections that provide different options and MIDI commands.

As an example play view has section [play.0] to work with song parts and [play.1] to work with loops inside one part. Also note that there may be multiple commands separated by colon (:) assigned to a button.

To know how looper commands are assigned to MIDI notes look at [MIDI controller]

Menu hints

In each view top row shows drum type and beats per minute. Just below it there are hints for available commands.
