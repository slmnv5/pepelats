Style drum

Drum patterns

Style drum is using text patterns specific for musical style and sample WAV files to produce drum accompaniment. There are plenty of musical styles, some of them linked on main page as examples.

Each style pattern is inside one section of INI file. Number of steps is equal to the longest drum length. Two examples of styles are below.

[Disco] – patterns for Disco

[Latin] – patterns for Latin

Drum names

Drum names used in pattern INI file and WAV file names are the same (ex. sd.wav, bd.wav):

  • bd, sd: are drum names (see below)
  • ".": silence on a step
  • "!": drum hit on a step
  • ac: Accent - increases drum hit volume on specific step
  • bd: Bass drum
  • sd: Snare Drum
  • lt: Low tom
  • mt: Medium tom
  • ht: High tom
  • ch: Closed hi-hat
  • oh: Open hi-hat
  • cy: Cymbal
  • rs: Rim shot
  • cp: Clap
  • cb: Cowbell
