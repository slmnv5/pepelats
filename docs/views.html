Views


A view combines looper state information on scree or web page and set of commands that may be processed in this state. There are 3 looper views and 3 menu configurations. The later are text file that configure available MIDI messages and respective commands. Also they contain view's description and command to display looper status.

  • * [play view] -- play / record / overdub / clear song parts and switching between them. Show loops for song parts. Silence, delete, re-arrange loops.

  • * [song_view] - delete, save, initialize or load song. Scroll list of saved songs. Select drum configuration.

  • * [service view] -- Restart looper, check for updates. Select drum type for loaded song, adjust drum volume and parameters.

More about this is in [menu_config]
