Views


A view combines looper state information on screen or web page and set of commands that may be processed in this state. There are 3 looper views and 3 related menu configurations. The later are text files that configure MIDI messages and commands. Also there is view description and command to display looper status.

  • [play.ini] -- view for playing and recording of song parts and loops

  • [song.ini] -- view for managing songs and drums

  • [serv.ini] -- view for restart and other service functions

More about this is in [menu_config]
