## Euclid drum

*Euclid drum* rythms use four parameters: steps, beats, offset and accent.

- Steps - length (or period) of the pattern.
- Beats - how many drum hits there is in one period.
- Offset - position of first beat, other beats are spread most evenly over all steps.
- Accent - how many steps have accented sound, e.g. if accent = 3 each 3rd step is accented if there is a beat.

Euclid drum allows creating long non-repeating patterns using simple notation. As an example 208 step long sequence
for bass drum (bd) and snare drum (sd):
~~~
bd : 16, 4, 0, 0
sd : 13, 8, 0, 0
~~~

This drum type is similar to IOS application Ruismaker which has great user manual explaining it all.
